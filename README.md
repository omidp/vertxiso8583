## vertx8583

This project is using JPOS and Vertx to parse [ISO 8583](https://en.wikipedia.org/wiki/ISO_8583) message over tcp network

##  Step to use it

* mvn clean install
* import into IDE
* run main class
* use telnet or netcat to send a message
* the result should be as follow
